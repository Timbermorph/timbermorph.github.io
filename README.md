# Jianzhe Liu — Personal Homepage

Source for [timbermorph.github.io](https://timbermorph.github.io/), built with plain HTML/CSS (no build step).

## Structure

- `index.html` — page content
- `style.css` — styling
- `images/profile.jpg` — profile photo

## Editing

Open `index.html` directly in a browser, or serve locally:

```
python3 -m http.server 8000
```

then visit `http://localhost:8000`.

## Deploying

This repo is a GitHub Pages user site: pushing to `main` on
`Timbermorph/Timbermorph.github.io` publishes automatically to
https://timbermorph.github.io/.
